Java 7 1/2: developer notes

What is the new Java strategy? Will the Lambda project manage to close the gap with scripts languages such as clojure and programming languages such as Scala? We will try to address these questions. Starting with a brief overview of Java SE 7, and in particular of the new features like the diamond and type infer and the Fork&Join framework, we will explore the Java SE 8 and, in particular, the long-awaited Lambda project aimed at delivering the functional programming support for Java necessary to start closing the gap with Scala.



Embedding Groovy in a Java application

In this talk I present my experience of embedding Groovy in Portofino 4 to support dynamic, run-time modifiable actions and business logic. We'll see in detail: classloading, performance and things to keep in mind to make Groovy scripts actually reloadable.

Speakers

Luca Vetti Tagliati

Luca Vetti Tagliati has more than 20 years of professional experience and holds a PhD from Birkbeck College (University of London). Over the years, he has been applying the OO design/programming, component-based and SOA in various sectors, ranging from smart cards to Private and Investment banking, mainly in Java SE/Java EE. After having worked for several years in the City of London for the major investment banks (Goldman Sachs, Lehman Brothers, Deutsche Bank, UBS, HSBC), he took up the position of Senior Lead Architect for a global banking group in Switzerland. Luca has worked closely with John Daniels (co-author of the book "UML Components") and Frank Armour (co-author of the book "Advanced Use Case Modeling"). He is the author of the book "UML e l’Ingegneria del Software: dalla Teoria alla Pratica" published in 2003 by HOPS Tecniche nuove.


Paolo Predonzani

Paolo Predonzani is co-founder of ManyDesigns and co-author of Portofino, the open source framework for enterprise applications. As a user and as a supplier, he has always sought a balance between features, time and architecture in enterprise software projects.
