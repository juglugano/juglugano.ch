Abstract

Exploring functional constructs using Java 8

With java8, more functional programming constructs / patterns can be realistically used in your code.
Let's explore a little bit what the functional world has to offer, with some java and haskell examples.

The talk will be presented in Italian or English language, depending on the participants.



Speaker

Sylvain Jermini

Sylvain has been writing programs and hacking for fun since 2000 and for profit since 2007 after he graduated from SUPSI (Manno). His main interests are in learning new and old programming languages, web technologies and "computer science" stuff in general.
