The Groovy Ecosystem

Groovy is the fastest growing JVM language out there. It might be because it is so easy for Java developers to pick it up, but also because there's an increasing number of projects and libraries that make use of Groovy as the starting point. In this workshop we'll cover frameworks, tools and libraries rooted in the Groovy language that can make your daily work a much more enjoyable experience. - Frameworks Grails, Griffon, Gaelyk - Build Systems Gant, Gradle - Testing Easyb, Spock - Code Quality CodeNarc, GMetrics - Libraries GPars, GContracts. Along with a slide deck there will be plenty of time for live code demonstration and on-hands experience. Attendees are encouraged to make this workshop as interactive as possible by steering the direction and choose how much time they would like to spend on a particular topic.

Speaker

Andres Almiray, Canoo Engineering AG

Andres is a Java/Groovy developer and Java Champion, with more than 12 years of experience in software design and development. He has been involved in web and desktop application development since the early days of Java. He has also been teacher of computer science courses in the most prestigious education institute in Mexico. His current interests include Groovy and Swing. He is a true believer of open source and has participated in popular projects like Groovy, Griffon, JMatter and DbUnit, as well as starting his own projects (Json-lib, EZMorph, GraphicsBuilder, JideBuilder). Founding member and current project lead of the Griffon framework.
